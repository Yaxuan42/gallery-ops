## 型号编码规则

Pierre Jeanneret 型号格式：`PJ-{类别}-{编号}-{变体}`

| 类别代码 | 含义 | 举例 |
|---|---|---|
| SI | Siège（座椅） | PJ-SI-59-A |
| BU | Bureau（桌/办公） | PJ-BU-19-A |
| TA | Table（桌） | PJ-TA-01-A |
| R | Rangement（收纳） | PJ-R-23-A |

变体字母 A/B/C/D/E 代表同一设计的不同版本或细节差异。

## 通用推断规则

当录入者提供的型号以 `PJ-` 开头时：
- **设计师** → Pierre Jeanneret
- **产地** → Chandigarh, India
- **产品分类** → Chandigarh
- **材质** → 默认 Teakwood（含 Cane 的型号需要录入者确认）

## 品相等级参考

| 等级 | 描述 |
|---|---|
| A | 品相优秀，原始状态保存完好，结构稳固，表面磨损极少 |
| B | 品相良好，有正常使用痕迹，可能经过专业修复，结构稳固 |
| C | 品相一般，有明显使用痕迹或修复痕迹，结构基本稳固 |
| D | 品相较差，需要修复，有结构性问题或大面积损伤 |

## 常见使用场景（Chandigarh）

Chandigarh 家具原用于勒·柯布西耶规划的印度昌迪加尔城市项目：
- 行政楼（Secretariat）
- 旁遮普大学（Punjab University）
- 高等法院（High Court）
- 立法议会（Legislative Assembly）
- 政府住宅（Government Housing）
